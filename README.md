# Prolix Director

A [loud employer](https://gitlab.com/fizzizist/loud-employer) but not quite

Primarily intending to be some "agentic" CLI tool as a potential replacement for Peter's initial bash script.

## Requirements

Noted in Loud Employer:

> Astonishingly, running `claude -p "/plugin install superpowers"` results in `Unknown skill: plugin`. So the only manual thing that needs to be done to set up this repo is, once you've run `docker compose up -d --build`, the first session, you'll have at install the `superpowers` plugin from the claude official marketplace.
